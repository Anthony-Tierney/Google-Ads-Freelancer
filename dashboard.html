CPA Achieved is cost ÷ conversions measured from the date the Target CPA was last changed up to yesterday. If the last change predates the available 30-day history (or there's no target), it's measured across the last 30 days. It's highlighted red with a warning icon when it's 20% or more below the campaign's Target CPA. Last Adjusted shows when the Target CPA was last changed — Google's change history only covers the last 30 days, so older changes show as "30+ days ago". Target CPA is read from the campaign's bid strategy; portfolio-level targets aren't included.
